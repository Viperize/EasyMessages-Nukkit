#EasyMessages
Send messages to your players with ease!

###Commands:
Main command: **easymessages**, **em**

|Sub-command|Description|
|-----------|-----------|
|**help**|Shows all EasyMessages commands|
|**message**, **m**|Sends a message|
|**motd**|Sets the server motd|
|**popup**, **p**|Sends a popup|
